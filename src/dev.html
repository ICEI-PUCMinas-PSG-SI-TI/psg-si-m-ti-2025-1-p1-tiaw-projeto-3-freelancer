Dentre as funções, estão algumas ferramentas para criação aleatória de usuários, serviços ou outras informações, isso agiliza o cadastro.

O proposito desta página é apenas testes, portanto na aplicação final, esta página será desabilitada ou limitada apenas para usuários administradores.
